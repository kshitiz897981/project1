# ExpenseTracker 💸

A modern and responsive Expense Tracker and Bill Splitter web application built using React and Tailwind CSS.

The application helps users manage personal expenses, visualize spending patterns, and split bills among groups efficiently.

---

# 🚀 Features Implemented

## 📊 Expense Dashboard

* Add new expenses
* Track expense amount, category, and date
* View all expenses in a clean dashboard layout
* Edit existing expenses
* Delete expenses instantly

---

## 📈 Spending Breakdown

* Dynamic category-wise expense visualization
* Animated custom bar chart
* Responsive chart design
* Hover tooltips for expense insights

---

## 👥 Bill Splitter

* Add and manage group members
* Remove members dynamically
* Enter amount paid by each participant
* Include/exclude members from split calculation
* Calculate fair bill distribution

---

## 🎨 UI/UX Features

* Responsive design for desktop and mobile
* Modern clean interface
* Smooth hover animations
* Tailwind CSS styling
* Interactive components

# 📂 Project Setup Instructions

## 1. Clone the Repository

```bash
git clone https://github.com/kshitiz897981/Expense-Tracker.git
```

---

## 2. Navigate to Project Folder

```bash
cd expense-tracker
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

The app will run on:

```bash
http://localhost:5173
```

---

# 🏗 Build for Production

```bash
npm run build
```
